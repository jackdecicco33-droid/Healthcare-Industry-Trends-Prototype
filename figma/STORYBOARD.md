# Website Storyboard for Figma

## Frame 1 — Landing / Hero
- Header navigation: Service lines, Resource library, Source trust, Original HTML
- Hero message: “Search resources by service line, level, and source type.”
- Summary metrics: service lines, resource cards, unique sources, consulting levels

## Frame 2 — Service Lines
- Six cards: Revenue Cycle, Supply Chain, Physician Enterprise, Quality, Clinical Optimization, Labor/Workforce
- Each card has “What this service line actually does” and “What consultants look for.”

## Frame 3 — Searchable Resource Library
- Search input at top
- Filters: service line, level, category
- Resource cards beneath
- Empty state when no matches are found

## Frame 4 — Source Trust
- Search sources
- Source cards by domain
- Credibility label and usage note

## Frame 5 — Mobile Version
- Stacked hero
- Single-column service cards
- Single-column resource cards
- Full-width filters
